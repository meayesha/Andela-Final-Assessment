## How to Stream, Use Tools, and Handle Images (Multimodal) with Chat History

This section demonstrates how to combine streaming, tool use, multimodal (image) input, and chat history in a single OpenAI chat session. This pattern is based on your agents folder and OpenAI documentation.

### 2. Define Tool Functions and Tool Map

```python
def record_user_details(email, name="Name not provided", notes="not provided"):
    print(f"Recording: {email}, {name}, {notes}")
    return {"recorded": "ok"}

def record_unknown_question(question):
    print(f"Unknown question: {question}")
    return {"recorded": "ok"}

tools = [
    {"type": "function", "function": {
        "name": "record_user_details",
        "description": "Record user details for follow-up.",
        "parameters": {

            "properties": {
                "email": {"type": "string"},
                "name": {"type": "string"},

        }

tool_map = {
```

### 4. Streaming with Tools and Multimodal

```python
openai = OpenAI(api_key=os.getenv("OPENROUTER_API_KEY"), base_url="https://openrouter.ai/api/v1")

def handle_tool_calls(tool_calls):
    results = []
    for tool_call in tool_calls:
        tool_name = tool_call.function.name
        arguments = json.loads(tool_call.function.arguments)
        fn = tool_map.get(tool_name)
        result = fn(**arguments) if fn else {"error": "unknown tool"}
        results.append({
            "role": "tool",
            "content": json.dumps(result),

    return results

done = False
```

### Limitations and Notes

- As of April 2026, OpenAI's Python SDK does not always return tool_calls in streaming chunks; you may need to make a non-streaming call to get tool_calls, then stream the next user/assistant turn.
- Multimodal (image) input is supported only in certain models (e.g., gpt-4o, gpt-4-vision-preview). Check OpenAI docs for the latest support.
- Streaming and tool use together is supported, but tool calls may only be available at the end of the streamed response.
- Always maintain the full `messages` history for context.
- For more, see the [OpenAI Python SDK docs](https://github.com/openai/openai-python) and [OpenAI API multimodal docs](https://platform.openai.com/docs/guides/vision).
